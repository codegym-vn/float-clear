Float and Clear Demo Page

Thuộc tính float

Mã nguồn

HTML
        
            <div class="outer">
                <button class="button-left">Float left
                <button class="button-right">Float right
                <div class="inner clear-block">
            <div>
CSS
        
            .outer {
                border: solid 1px black;
                height: 100px;
                width:500px;
            }
            .button-left{
                float:left;
            }
            .button-right{
                float: right;
            }

Thuộc tính clear

Mã nguồn

HTML
        
            <div class="outer">
                <button class="button-left">Float left
                <button class="button-right">Float right
                <div class="inner clear-block">
            <div>
CSS
        .outer {
            border: solid 1px black;
            height: 100px;
            width:500px;
        }
        .inner{
            float: left;
            height: 80px;
            width: 200px;
            background-color: yellow;
        }
        .button-left{
            float:left;
        }
        .button-right{
            float: right;
        }
        .clear-block{
            clear: both;
        }
